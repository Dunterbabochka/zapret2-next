# Launch checklist

- Publish `v0.1.0` as a draft release only after Windows 10 and Windows 11 smoke tests.
- Record a short GIF showing: `general.bat` start, `service.bat` status, preset selection, and diagnostics. Do not show a username or local path.
- Add the GIF to the README only after the recorded build matches the release SHA256.
- Publish a measured compatibility table containing provider/region, Windows version, strategy, YouTube web/video, Discord web/voice/screen share, and test date.
- Use repository topics: `zapret2`, `windows`, `windivert`, `discord`, `youtube`, `dpi`.
- Describe the project as an independent Zapret 2 Windows bundle; do not imply upstream endorsement or use an outage/takedown claim that cannot be verified at publication time.
